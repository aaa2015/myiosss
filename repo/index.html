SS Apple Repository

About

This is a Cydia/Sileo repository for SS (Socket Statistics) - a Linux ss command implementation for iOS.

Features:

  • Display TCP/UDP socket statistics
  • Show process information (-p)
  • High performance (23x faster than alternatives)
  • Linux ss compatible output format

Add Repository

Usage

After installation, use the ss command in terminal:

# Show listening TCP/UDP sockets with process info
ss -tulnp

# Show established connections
ss -t state established

# Show help
ss -h

Requirements

  • iOS 13.0 or later
  • Jailbroken device
